## 作业一：
### 要求：
        编程序实现四种以上测试用例的自动生成；
        以三角形问题为例，
        (1)能够接收每一个变量的值域的输入.
        (2)自动生成测试用例.
        (3)并输出预期结果
        (4)操作文档
        
### 实现细节：
一、编程环境和开发语言  
1、开发环境：eclipse + Window64 + 64位EXE4J软件 + 64位jre

3、可运行的环境：任意平台
